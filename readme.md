## 🧠 Why it matters

Most documentation systems fail because they are:
- Static  
- Redundant  
- Hard to maintain  

Supradoc takes a different approach:
> Documentation should behave more like **code** — reusable, dynamic, and reliable.

---

## 🧩 Core Product: Suprasync

**Suprasync** is our first product — a Confluence Cloud plugin that introduces **dynamic content management** through reusable variables.

### Key Features
- 🔤 Text variables  
- 🔗 Link variables  
- 💻 Code variables (with syntax highlighting)  
- 🖼️ Image variables  
- 👤 User reference variables  
- 🏷️ Tag variables  
- 📅 Date variables  
- 📦 Archiving system  
- 📊 Usage statistics  

This enables:
- Instant updates across documents  
- Reduced duplication  
- Consistent, error-free documentation
